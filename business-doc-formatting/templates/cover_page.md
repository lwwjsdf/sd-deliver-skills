# Cover Page Template

## Layout

```
┌─────────────────────────────────────────┐
│                                         │
│            [Company Logo]               │
│              (2cm from top)             │
│                                         │
│                                         │
│          Document Title                 │
│          ==============                 │
│              20pt Bold                  │
│                                         │
│         Subtitle / Project Name         │
│              14pt Regular               │
│                                         │
│                                         │
│          Version: 1.0                   │
│          Date: 2024-01-15               │
│              11pt Regular               │
│                                         │
│                                         │
│                                         │
│                                         │
│                                         │
│         CONFIDENTIAL                    │
│         --------------                  │
│              10pt Bold Red              │
│                                         │

```

## Markdown Example

```markdown
<div align="center">

![Company Logo](assets/logo.png)

# System Non-Functional Requirements

## BEA CDP + SF Project

**Version:** 1.0  
**Date:** 2024-06-05  
**Prepared by:** SensorsData Delivery Team

---

**CONFIDENTIAL**

*This document contains proprietary and confidential information. Unauthorized distribution is strictly prohibited.*

</div>
```

## Word Formatting Instructions

1. Insert → Picture → Logo (center, 3cm width)
2. Press Enter 3 times
3. Type title, format: 20pt, Bold, Center
4. Press Enter 2 times
5. Type subtitle, format: 14pt, Center
6. Press Enter 4 times
7. Type version/date, format: 11pt, Center
8. Press Enter until 3cm from bottom
9. Type "CONFIDENTIAL", format: 10pt, Bold, Red, Center
10. Page Layout → Page Borders → Box, 0.5pt, 1cm from edge (optional)

## Style Settings

| Element | Font | Size | Bold | Color | Align |
|---------|------|------|------|-------|-------|
| Logo | - | - | - | - | Center |
| Title | Calibri | 20pt | Yes | Black | Center |
| Subtitle | Calibri | 14pt | No | Black | Center |
| Meta | Calibri | 11pt | No | Gray | Center |
| Classify | Calibri | 10pt | Yes | Red | Center |
