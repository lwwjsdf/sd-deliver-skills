## 快速参考：常见数据问题类型

| 问题类型 | 示例 | 修复位置 |
|----------|------|----------|
| 枚举值不全 | 地域只有 mainland | `region_distribution` |
| 比例失衡 | L4 用户占 40% | `user_segments.*.ratio` |
| 数值类型错误 | amount 是 int 不是 float | 事件属性生成逻辑 |
| 缺少事件 | 没有退款流程 | `event_sequences` 新增序列 |
| 属性缺失 | 支付事件缺少 pay_method | 事件属性定义 |
| 时间异常 | 事件时间戳集中在同一小时 | 时间分布逻辑 |
| ID 格式错误 | distinct_id 包含中文 | ID 生成规则 |
